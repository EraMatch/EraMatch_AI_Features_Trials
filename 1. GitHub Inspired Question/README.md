# GitHub Inspired Question Trials

This directory contains trials and experiments for the ACE-Recruiter engine, focusing on assessing candidate profiles using GitHub-inspired questions and performance rubrics.

## Overview

The experiments here trace the evolution of the screening system, evaluating the performance and accuracy of different parallelization strategies and LLM integrations. 

## Files

- [**performance_history.md**](./performance_history.md): Tracks the performance optimization history for the ACE-Recruiter, detailing the trial phases (from Baseline to Lightning Parallelism), latency, and the architectural shifts.
