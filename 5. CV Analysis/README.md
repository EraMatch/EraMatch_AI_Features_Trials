# CV Dataset Generation

## Notebooks

### v3 — `cvs-eramatch_generate-dataset-v3.ipynb`
Original benchmark dataset. Generates 10,000 synthetic CVs across 5 tiers (T1–T5) and 10 IT domains with CVSchema v3.0 ground truth. Monolithic notebook. Output: PDFs + JSONs.

### v4 — `cvs-eramatch_generate-dataset-v4.ipynb`
Enhanced pipeline with modular architecture. Key changes from v3:
- 22 HTML templates across T1–T5 (vs limited templates)
- Realistic GitHub project generation and domain-specialization email mapping
- Separated modules: `cv_generator_batch1.py` (premium), `cv_generator_batch2.py` (synthetic), `pdf_renderer.py`, `schema.py`, `specialization_mapper.py`, `github_projects.py`
- Validation script (`validate_batch1.py`)
- Master metadata CSV tracking all generated CVs
